Política de Privacidade – Conserlar

A Conserlar valoriza a privacidade de seus usuários e está comprometida em proteger as informações coletadas por meio do aplicativo Conserlar. Esta Política de Privacidade descreve como os dados são coletados, utilizados e protegidos.

1. Informações Coletadas

O aplicativo Conserlar pode coletar os seguintes dados:

1.1 Dados pessoais fornecidos pelo usuário

  • Endereço de e-mail, quando informado voluntariamente pelo usuário.

1.2 Dados coletados automaticamente

  • Identificadores do dispositivo (ID do dispositivo).
  • Informações técnicas, como modelo do aparelho e versão do sistema operacional.

2. Uso das Informações

  • Permitir o funcionamento correto do aplicativo.
  • Identificar o usuário quando necessário.
  • Melhorar estabilidade, segurança e desempenho.
  • Comunicação com o usuário, quando aplicável.

3. Compartilhamento de Dados

A Conserlar não compartilha, vende ou aluga dados pessoais a terceiros.

4. Permissões do Aplicativo

  • Armazenamento: acesso a arquivos e documentos.
  • Bluetooth: comunicação com dispositivos externos.
  • Internet: funcionalidades online.

5. Segurança das Informações

São adotadas medidas técnicas razoáveis para proteger os dados contra acesso não autorizado.

6. Privacidade de Crianças

O aplicativo não é destinado a menores de 13 anos.

7. Retenção e Exclusão de Dados

O usuário pode solicitar exclusão dos dados a qualquer momento.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente.
